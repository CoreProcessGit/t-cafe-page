Privacy Policy

Last Updated: April 2026

1. Introduction

This Privacy Policy describes how COREPROCESS ("we", "us", "our") collects, uses, and protects information through the T-CAFE application ("the App"), a test management plugin for Atlassian Jira.

By installing and using the App, you agree to the practices described in this Privacy Policy.

2. Information We Collect

2.1 Data from Jira

The App accesses the following data from your Jira instance through Atlassian Forge APIs:

  • User Information: Display name, account ID, email address, and avatar URL of Jira users within the project
  • Project Information: Project key, project name, and project configuration
  • Issue Information: Issue key, summary, status, type, and assignee (for defect linking features)

2.2 Data Created Within the App

Users create and manage the following data through the App:

  • Test Cases: Name, description, precondition, priority, status, test steps, and custom fields
  • Test Plans: Name, description, schedule, and test case assignments
  • Test Reports: Execution results, pass/fail status, and related metrics
  • Factor Combinations: Factor definitions, constraints, simulation results, and assignments
  • Attachments: Files uploaded by users (stored in each user's Atlassian Forge Storage, not on our servers)
  • Board Posts: Announcements and notices created by administrators

2.3 Data We Do NOT Collect

  • We do not collect personal data beyond what is available through Jira's standard APIs
  • We do not collect payment or financial information
  • We do not use cookies or tracking technologies
  • We do not collect data from users outside of the Jira environment

3. How We Use Information

We use the collected information solely for the following purposes:

  • Providing App Functionality: Creating, managing, and executing test cases, test plans, and test reports
  • User Management: Assigning roles and permissions within the App
  • Defect Tracking: Linking test results to Jira issues for defect management
  • Reporting: Generating test execution reports and analytics
  • Data Display: Showing user names and avatars in the App's interface

We do NOT use your data for advertising, profiling, selling to third parties, or training AI/ML models.

4. Data Storage and Security

4.1 Storage Location

  • Test data: Stored on COREPROCESS's private database server located in South Korea
  • Attachments: Stored in each user's Atlassian Forge Storage (managed by Atlassian, not on our servers)
  • Temporary upload data: Processed through Atlassian Forge Storage and deleted after processing

4.2 Security Measures

  • Encryption: Sensitive test data encrypted using AES-256-CBC
  • Access Control: Role-based access control (RBAC) with four levels
  • API Security: API key authentication for all server communication
  • Input Sanitization: DOMPurify for XSS prevention
  • SQL Injection Prevention: Parameterized database queries
  • Transport Security: HTTPS/TLS encryption for data in transit

4.3 Data Isolation

Each Jira Cloud instance's data is isolated using a combination of cloud_id and project_key. Data from one instance cannot be accessed by another.

5. Data Sharing

We do NOT share your data with any third parties, except:

  • Atlassian: Data transmitted through Atlassian's Forge platform as required for functionality
  • Cloudflare: Network traffic passes through Cloudflare's tunnel service for secure connectivity

We will disclose information if required by law or legal process.

6. Data Retention

  • Active data: Retained as long as the App is installed and active
  • Deleted data: Permanently removed from our database when deleted within the App
  • App uninstallation: You may request complete deletion by contacting us

7. Your Rights

You have the right to:

  • Access: View all stored data through the App's interface
  • Correction: Edit or update any data through the App
  • Deletion: Delete individual records or request complete data removal
  • Export: Export data in CSV, JSON, or Excel format
  • Portability: Request a copy of all data in a standard format

To exercise these rights, contact us at cp.content100@gmail.com.

8. International Data Transfers

Your data is stored on servers located in South Korea. We ensure appropriate security measures are in place regardless of your location.

9. Children's Privacy

The App is designed for enterprise Jira environments and is not intended for children under 16.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Significant changes will be communicated through the App or via email.
