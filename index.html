Features

Everything you need for test management, integrated directly into Jira.

Test Case Management

Create and manage test cases with Single and Factor Combination types. Support for test steps, preconditions, priorities, and custom fields.

Test Plan & Execution

Organize test cases into test plans. Track execution status with Pass, Fail, Block, and Skip results. Monitor progress in real-time.

Test Reports & Analytics

Generate comprehensive test reports with execution statistics, pass/fail rates, and progress tracking. Export reports in CSV, JSON, or Excel.

Factor Combination Testing

Generate test combinations using Full Combination, Pairwise, or Mixed algorithms. Apply constraints and simulate test coverage efficiently.

Jira Issue Integration

Create and link Jira issues directly from test execution. Attach screenshots with built-in image editor for defect reporting.

Attachment Management

Upload, download, and manage test artifacts. Support for chunked file upload with progress tracking and drag-and-drop interface.

Configuration

Customize priorities, case types, components, plan statuses, and user-defined fields. Tailor the app to your team's workflow.

User Permissions

Role-based access control with Admin, Team Admin, Tester, and Developer roles. Control who can create, edit, and execute tests.

Multi-Language Support

Full internationalization with English, Korean, and Japanese. Automatic language detection based on Jira settings.
